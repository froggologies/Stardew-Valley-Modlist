# Stardew Valley Modpack

This is a comprehensive and carefully curated modpack for Stardew Valley designed to enhance the game's visuals, quality of life, and immersion while maintaining the vanilla feel. 

**What's inside this pack?**
- **🎨 Visual & Aesthetic Overhauls:** Completely revitalizes the look of the game with vibrant map recolors, cuter seasonal characters, new building textures, realistic water/lighting, and adorable replacements for your farm animals and pets.
- **✨ Quality of Life (QoL) Enhancements:** Major gameplay improvements to reduce tediousness without feeling like cheats. Includes an upgraded crafting system, detailed UI tooltips (like daily luck, crop growth, and gift tastes), and automatic gates.
- **🌍 Immersion & Atmosphere:** Makes the world feel more alive with wind effects, visible fish swimming in the water, dynamic reflections, and even voice acting for love interests.
- **⚙️ Gameplay & Mechanics Tweaks:** Tools to tweak the game to your pacing, like full control over time (speed up, slow down, or freeze), sitting to regenerate stamina, and revamped clothing mechanics.
- **🛠️ Easy Script Loaders:** Custom Linux-native bash scripts that allow you to seamlessly switch between different mod profiles (e.g., "Daily Use" vs "Multiplayer").

## Requirements
- [SMAPI](https://smapi.io/) is required to use these mods.

## How to use this modpack

### Linux Users
The script loaders in this modpack are built and tested specifically for Linux. 
1. `git clone` this repository directly into your main Stardew Valley game folder so that you can utilize the provided scripts.
2. Navigate to the `scripts` directory and execute one of the `.sh` scripts to apply a specific mod preset. The `.csv` files act as your presets, defining exactly which mods should be active, while the `.sh` scripts do the work of applying those presets:
   - `apply_mods.sh`: Applies the **"All Mods"** preset using `mods.csv` (which is auto-generated by `generate_csv.sh`).
   - `apply_default.sh`: Applies the **"Daily Use"** preset using `mods_default.csv`.
   - `apply_multiplayer.sh`: Applies the **"Multiplayer"** preset using `mods_multiplayer.csv`. 
     *(Note: Currently, **[CP] Visual Enemy Variety** and **MonsterVariety** are disabled in this preset because they can cause crashes in the mines during multiplayer sessions.)*

#### Notes:   
- *You can easily customize these presets by opening their respective `.csv` files and changing the status of any mod to `TRUE` or `FALSE`.*
- *It is also possible to add these scripts as a "Non-Steam Game" to Steam for easier launching.*

### Mac & Windows Users
I am not sure if the loader scripts will run on Mac.
For Mac and Windows users, you can bypass the scripts:
1. Navigate to your `Mods` directory (`cd Mods`).
2. Clone this repository directly into your current `Mods` directory by running:
   ```bash
   git clone https://github.com/froggologies/Stardew-Valley-Modlist.git .
   ```
